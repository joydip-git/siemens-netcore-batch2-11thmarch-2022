create an application to find out whether a person is outstanding or not.

Library application:
Author
	data - name and books published
	method - IsOutstanding
			 args: NA
			 return type: bool
an author is outstanding if he/she has published more than 5 books

Trainee
	data - name and marks
	method - IsOutstanding
			 args: NA
			 return type: bool
a student is outstanding if he/she has scored more than 85

console application:
In Main method:
create objects of both author and trainee (at least two each)
store them in an array
iterate through them again and print the name of ONLY outstanding people
